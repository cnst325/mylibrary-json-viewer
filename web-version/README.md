# MyLibrary JSON Viewer - Web Version

A lightweight, browser-based viewer for MyLibrary app backup files. No installation required.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Try%20Now-brightgreen)](https://cnst325.github.io/mylibrary-json-viewer/web-version/)

## Features

- **Browser-Based** - No installation, just open and use
- **Privacy First** - All data processed locally in your browser
- **Cross-Platform** - Works on Windows, Mac, Linux, and mobile devices
- **Multilingual** - English, Korean, Japanese, German
- **Export to CSV** - Excel-compatible format

## Quick Start

### Online (Recommended)

Visit: [https://cnst325.github.io/mylibrary-json-viewer/web-version/](https://cnst325.github.io/mylibrary-json-viewer/web-version/)

1. Click "Open JSON File" button
2. Select your MyLibrary backup JSON file
3. Browse and export your library data

### Offline

Download the files and double-click `index.html` to open in your browser.

## What You Can View

- **Collection** - Books, eBooks, audiobooks, and other media
- **Loans** - Active and historical loan records
- **Borrowers** - Contact information
- **Wishlist** - Items you plan to acquire
- **Locations** - Physical storage locations

## Browser Compatibility

- Chrome/Edge 90+
- Firefox 88+
- Safari 14+

## Privacy & Security

All data processing happens entirely in your browser. No data is ever sent to any server.
